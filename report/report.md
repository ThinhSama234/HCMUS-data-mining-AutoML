# AutoML Framework Benchmark — Report (DRAFT)

Reproduction of a fair AutoML comparison following *AMLB: an AutoML Benchmark*
(Gijsbers et al., JMLR 2024). **Status: MVP (User Story 1) demonstrated end-to-end** on a
smoke suite; the full, fair 3-framework thesis run is pending (Docker / Linux cloud).

## MVP smoke result (3 datasets, 1 fold, 60s budget, local CPU)

Suite: credit-g (binary), vehicle (multiclass), Moneyball (regression). One identical
protocol for every runner. Overall average rank (1 = best):

| Framework | Avg rank |
|---|---|
| flaml | 1.33 |
| RandomForest (baseline) | 1.67 |
| constantpredictor (baseline) | 3.00 |

By task type: flaml wins multiclass + regression; RandomForest wins binary; both beat the
constant baseline on every task. Per-dataset scores are in `results/`.

This validates the pipeline end-to-end: **AMLB → results.csv → analysis/ → ranking**, under
one identical protocol.

## Platform note (drives the full run)

On this macOS arm64 / Python 3.9 box, 4 of 6 runners failed to install/run **locally**:

- **TunedRandomForest** — `sklearn 1.6.1` removed `_check_fit_params`, which AMLB's code imports (version drift; needs an older sklearn pin).
- **flaml** — xgboost needed `libomp` (fixed with `brew install libomp`; now runs).
- **H2OAutoML**, **AutoGluon** — Linux-first setup scripts fail on macOS (`AutoGluon setup.sh` exit 127).

AMLB is Linux-first and ships **Docker mode** for exactly this. The fair comparison of
H2O / FLAML / AutoGluon must run in **one consistent environment** — `-m docker` (prebuilt
Linux images) or a Linux/cloud box (User Story 5). The failure run is preserved at
`results/_smoke_with_failures.csv` (real data for the US2 failure-capture analysis).

## Pitfalls avoided (FR-012 / SC-006) — to expand

- Unequal budgets → one identical constraint applied to every runner.
- Inconsistent metrics → auc / neg_logloss / neg_rmse, by task type.
- Ignored failures → failures captured + categorized (US2; see `_smoke_with_failures.csv`).
- No environment isolation → Docker for the full run (US5).

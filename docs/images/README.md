# Screenshots

UI captures used by the README and docs. Refresh them after UI changes:

1. Run the console: `streamlit run console/app.py` (→ http://localhost:8501).
2. Capture each page full-height (viewport ~1400px wide) into this folder:
   `evaluation.png`, `datasets.png`, `methods-catalog.png`, `training.png`.

Keep each file under ~300 KB.
